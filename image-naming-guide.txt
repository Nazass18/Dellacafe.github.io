# Image Requirements:
- Format: JPG
- Quality: High-resolution, well-lit
- File size: Optimize to under 500KB
- Color profile: sRGB
- Compression: Web-optimized

# Naming Conventions:
1. Use all lowercase letters
2. Use hyphens (-) to separate words
3. Be descriptive but concise
4. Don't use spaces or special characters
