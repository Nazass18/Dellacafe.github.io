# Decallafe - Premium Cafe Website

A modern, responsive website for Decallafe, a premium cafe specializing in desserts and drinks. The website features a clean, elegant design with full e-commerce functionality.

## Features

- Responsive design that works on all devices
- Interactive menu system with cart functionality
- Beautiful animations and transitions
- User-friendly navigation
- Special offers and promotions section
- Contact information and location map
- Newsletter subscription
- Social media integration

## Pages

1. Home
2. Drinks Menu
3. Sweet Desserts Menu
4. Savory Desserts Menu
5. About Us
6. Shopping Cart

## Technology Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome Icons
- Google Maps Integration

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/decallafe-website.git
```

2. Navigate to the project directory:
```bash
cd decallafe-website
```

3. Open `index.html` in your web browser to view the website.

## Required Images

The website requires various images for the menu items and design elements. Please ensure you have the following image directories set up:

- `images/about/` - For team photos and cafe interior
- `images/drinks/` - For beverage menu items
- `images/sweet/` - For sweet dessert menu items
- `images/savory/` - For savory menu items
